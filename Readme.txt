# 🚗 Yasin Motors – Car Rental Web Portal

Yasin Motors is a **full-stack web application** designed for streamlined car rental operations. Developed as a final-year TYBSc Computer Science project, it allows users to rent vehicles based on either **daily rates or kilometer-based pricing**, and includes a powerful **admin dashboard** for full control over vehicles, bookings, drivers, and analytics.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Bootstrap
- **Backend:** PHP (Core PHP)
- **Database:** MySQL
- **Additional:** JavaScript, jQuery, AJAX

---

## 🚀 Features

### 👤 Customer Panel
- Secure Login / Registration / Forgot Password
- Book cars by **date** or **distance (KM)**
- View and manage personal bookings
- Optional **driver selection**
- Location-based pickup and return
- Submit feedback/testimonials

### 🔧 Admin Panel
- Dashboard with key stats (users, vehicles, bookings, etc.)
- Manage:
  - 🏎️ Vehicles
  - 🚘 Brands
  - 👨‍✈️ Drivers
  - 📍 GPS-based vehicle tracking
  - 🔄 Maintenance status
- View and respond to contact queries
- Export analytics and reports
- Activate/deactivate customer subscriptions

---

## 📦 Database Structure (Highlights)

- `tblusers`: Stores user details  
- `tblvehicles`: All vehicle metadata  
- `tblbooking`: Booking records (daily & KM-based)  
- `tbldriver`: Driver info and availability  
- `tblbrands`, `tblcontactusinfo`, `tbltestimonial`, `tblsubscriber`, etc.

---


## 📍 Booking Workflow

1. **User logs in**
2. Selects car & booking type (daily or kilometer)
3. (Optional) Selects a driver
4. Enters pickup and return location
5. Confirms booking
6. Real-time updates and feedback options enabled
